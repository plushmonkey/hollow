# HollowOS
Toy x64 operating system for learning.  

## Requirements
Compilation and ISO creation requires Linux/WSL.  
Emulation can be done on any system that has qemu.  

### Compilation
- build-essential
- nasm
- clang
  
### ISO creation
- mtools
- grub-common
- grub-efi
- xorriso
  
### Emulator
- qemu
